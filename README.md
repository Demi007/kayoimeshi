# 通い飯

## サイト概要
食事処紹介及びレビューサイト

### サイトテーマ
落ち着いて食事がしたい、自分だけの行きつけが欲しい、そんな人たちへ

### テーマを選んだ理由
人が食事に求めるものはなんでしょう？美味しさはもちろんですが、目新しさや懐かしさ、シチュエーションも人それぞれだと思います。このサイトは、落ち着いて食事できるところが知りたい、会社帰りに一人で、もしくは同僚と、ゆっくりと食事

### ターゲットユーザ
主に30代~50代の男性
